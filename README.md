# KiraStudio Public Repository

This is the public facing repo of [KiraStudio](https://kirastudio.org/). It contains the localization files, documentation, and the Windows releases of the app. Feel free to correct typos in the doc, to contribute a new translation or report a bug. You can also join our [Discord Server](https://discord.gg/Pbzw2Cwsb6) to discuss the app further.
